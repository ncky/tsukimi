# Tsukimi Dark Preload (Chrome)

Small helper extension to reduce the “white flash” on page load by applying a Tsukimi-dark background at `document_start`.

## Install

1. Open `chrome://extensions` (or `brave://extensions`, `edge://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select `apps/chrome/tsukimi-dark-preload/`.

## Notes

- This runs on **all sites** by default. If you want to scope it, edit `manifest.json` and narrow `matches` / `host_permissions`.

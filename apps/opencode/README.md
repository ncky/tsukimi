# opencode

Tsukimi theme for OpenCode.

## Install

Copy `tsukimi.json` to one of:

- `~/.config/opencode/themes/tsukimi.json` (user-wide)
- `<project>/.opencode/themes/tsukimi.json` (project-specific)

Then select it with `/theme` (pick `tsukimi`), or set it in `opencode.json`:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "theme": "tsukimi"
}
```

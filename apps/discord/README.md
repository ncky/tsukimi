# discord

Tsukimi Discord theme (custom CSS).

## Recommended: Vencord

Vencord supports loading local themes and editing “QuickCSS”.

1. Install Vencord.
2. Open Discord → **User Settings** → **Vencord** → **Themes**.
3. Use **Open Themes Folder** and copy `tsukimi-dark.theme.css` into that folder.
4. Enable the theme in the Vencord Themes list.

If you prefer QuickCSS instead:

1. Open Discord → **User Settings** → **Vencord** → **Themes** → **Open QuickCSS**.
2. Paste the contents of `tsukimi-dark.theme.css`.

## Notes

- This targets Discord’s dark theme.
- Vencord docs: https://vencord.dev/

# kvantum

This folder contains the Tsukimi Kvantum theme files.

## Install

Kvantum themes are typically installed as theme folders under `~/.config/Kvantum/`.

- Dark: create a theme folder (e.g. `TsukimiDark`) and place a `.kvconfig` + `.svg` inside it.
- Light: create a theme folder (e.g. `TsukimiLight`) and place a `.kvconfig` + `.svg` inside it.
